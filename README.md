# Sign Language Detection using YOLO

This project implements **sign language detection** using the **YOLO object detection model**. It trains a custom YOLO model on annotated hand sign images to detect and recognize sign language gestures in real time.

## Features

* YOLO-based hand sign detection
* Custom annotated dataset
* Training and inference using Jupyter Notebook
* Supports image, video, and webcam input

## Tech Stack

* Python
* YOLO (Ultralytics)
* OpenCV

## Usage

1. Install dependencies
2. Open `yolo.ipynb`
3. Train the model on the dataset
4. Run inference on images or live camera

## Dataset

Annotated sign language images in YOLO format.
